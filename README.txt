Wii Menu Linker v1.0
====================

¡Gracias por instalar Wii Menu Linker!

Esta aplicación te permite crear un menú interactivo estilo Wii para lanzar tus juegos y programas favoritos desde Wallpaper Engine.

INSTRUCCIONES DE USO:
---------------------
1. Abre "Wii Menu Linker" desde el menú inicio o el acceso directo del escritorio.
2. Configura tus slots con juegos, programas o enlaces web.
3. Guarda tu layout personalizado.
4. Usa la opción "Generate Config File" para crear scripts compatibles con Wallpaper Engine.
5. Puedes ejecutar la app en segundo plano usando el acceso directo "(Hidden)" o el archivo `run_hidden.vbs`.

REQUISITOS:
-----------
- Windows 10/11 (64-bit)
- No requiere instalación de Node.js, ya viene incluido.

CARACTERÍSTICAS:
----------------
- Interfaz estilo Wii con 12 slots configurables.
- Detección automática de juegos Steam y Epic Games.
- Búsqueda en el menú inicio de Windows.
- Generación de scripts para Wallpaper Engine.
- Ejecución en segundo plano y opción de inicio automático.

SOPORTE:
